# Ideas (Out of Scope for MVP)

## ML / AI
- NLP symptom extraction and categorization from transcriptions
- Severity prediction based on language patterns
- Anomaly detection on wearable time-series data
- AI-generated visit summaries for doctors

## Wearable Integrations
- Apple HealthKit / Google Health Connect real data
- Continuous heart rate monitoring with symptom correlation
- Sleep quality tracking integration
- Activity level context for symptom entries

## Features
- Audio recording + playback (not just transcription)
- Photo/video attachment to symptoms
- Medication tracking + interaction warnings
- Symptom pattern alerts (escalation detection)
- Multi-patient dashboard for clinicians
- FHIR/HL7 integration for EHR systems
- Offline-first with service worker
- Push notifications for logging reminders

## UX
- Body map for precise pain location
- Symptom templates / quick-log presets
- Mood tracking overlay
- Calendar heatmap view
